选择卡片 — 两张大卡片，选完才进上传
点击「空间」Tab后，先全屏展示室内/室外选择。用户明确点选后，卡片收起并出现上传区域。
决策优先，信息传递最清晰，适合首次使用用户。
✓ 最清晰，两个选项对比明确 ✓ 天空盒差异可直观展示 ✗ 多一次点击 ✗ 熟练用户觉得多余

方案 B
分步引导 — 步骤指示器，类型选择和上传各一步
进入空间Tab后出现步骤指示器：第一步选室内/室外（两个横排按钮），选完后点「下一步」进入第二步上传。
流程感强，适合需要强引导的场景；步骤指示器也让用户知道还有几步。
✓ 流程明确，操作不会出错 ✓ 可后续增加更多步骤 ✗ 增加一次「下一步」点击 ✗ 轻量任务感觉流程偏重

方案 C
Toggle 切换 — 上传区上方一个横向开关
进入空间Tab后直接出现上传区，顶部一个「室内 | 室外」Toggle。上传区和引导文案随切换实时更新，无需额外点击。
操作最少，和「物体」Tab下的体验保持一致，熟练用户最友好。
✓ 零额外点击，直接上传 ✓ 与整体设计风格最一致 ✗ 默认值若错误用户可能不注意 ✗ 天空盒差异没有前两种直观
