## 竞品详情

### World Labs — Marble

**官网**：https://www.worldlabs.ai / https://marble.worldlabs.ai  
**简介**：Fei-Fei Li 创立的空间智能公司，产品 Marble 是纯 AI 生成式 3D 世界平台——输入文本、图片、视频或 360 全景，直接生成可探索、可编辑的 3D 环境，输出 3DGS 格式，无需用户采集。

#### 核心功能

| 功能 | 是否有 | 实现质量 | 用户评价 |
|------|--------|---------|---------|
| 手机拍摄扫描/采集 | ✗ | — | — |
| AI 从文本/图片生成 3D | ✓ | 好（Marble 1.1+ 大幅改善）| 视觉质量高，但有时不一致 |
| 从视频生成 3D | ✓ | 中（仍有artifact）| — |
| Web Viewer 分享 | ✓ | 好（实时浏览器渲染）| — |
| 3D 编辑（扩展/修改世界）| ✓（Standard+）| 中 | 功能尚在成熟 |
| 导出（3DGS/Mesh）| ✓（Pro+）| — | Pro 才能导出高质量 Mesh |
| API 接入 | ✓ | 好 | 独立计费，$1/1250 credits |
| 团队协作 | ✓（Enterprise）| — | — |
| 移动端查看 | ✓ | 好（浏览器内）| — |
| 实景重建（照片→精确3D）| ✗ | — | 生成式，非精确重建 |

#### 订阅定价（Marble 应用端）

| 层级 | 包含生成次数 | 主要功能 |
|------|-----------|---------|
| Free | 4次 | 文本/单图/360全景生成，只能查看 |
| Standard | 12次/月 | 多图/视频输入，编辑，导出，下载社区内容 |
| Pro | 25次/月 | 世界扩展，高质量 Mesh 导出，商用授权 |
| Max | 75次/月 | 高产量生产用途 |
| Enterprise | 定制 | 大团队，灵活方案 |

**API 计费**：$1 = 1250 credits；标准模型生成一个 3D 世界约 1500-3100 credits（$1.2-2.5）

#### 用户真实评价

**正面反馈**：
- "从一张图生成的世界视觉效果令人惊艳"
- "浏览器实时渲染很流畅，不用下载任何东西"
- "3DGS 技术选型很聪明，比 NeRF 渲染快很多"

**负面反馈**：
- "生成的世界无法精确还原现实场景，更像风格化创作"
- "Free 版 4 次生成明显不够体验"
- "不能从真实空间拍摄重建，无法用于建筑/房产场景"

#### 优弱势分析

| 核心优势 | 核心弱势 |
|---------|---------|
| AI生成质量领先，Fei-Fei Li背书，融资充足 | 纯生成式，无法精确重建真实空间 |
| 浏览器实时渲染，无需安装 | 免费版生成次数极少（4次）|
| API生态完整，可集成 | 编辑能力仍在成熟，难以精细控制 |
| 支持多模态输入（文本/图/视频/360）| 对建筑/文旅/房产场景还原度不足 |

#### 差异化切入点

> 相对于 World Labs，我们的机会：**真实空间高保真重建**。Marble 是"凭空创造世界"，Aholo3D 是"精确数字化现实世界"。两者目标用户和场景互补，但对需要精确还原（量房、文物存档、现场保全）的用户，Aholo3D 有明确优势。

### Polycam

**官网**：https://poly.cam  
**简介**：最流行的移动端 3D 扫描应用，支持 LiDAR（iPhone/iPad Pro）+ 普通相机摄影测量 + 3DGS 采集，覆盖 iOS/Android/Web，功能最全面，用户基数最大。

#### 核心功能

| 功能 | 是否有 | 实现质量 | 用户评价 |
|------|--------|---------|---------|
| 手机拍摄扫描/采集 | ✓ | 好（LiDAR）/ 中（普通相机）| 有LiDAR时效果明显更好 |
| 3DGS 采集与处理 | ✓ | 中-好 | 云端处理，速度可接受 |
| AI 生成 3D | ✓（有限）| 中 | 非核心，效果一般 |
| Web Viewer 分享 | ✓ | 好 | 可嵌入、可分享链接 |
| 3D 编辑 | ✗（极有限）| — | 基本没有编辑能力 |
| 楼层平面图生成 | ✓（Business+）| 好 | 专业用户高频需求 |
| 15+ 格式导出 | ✓（Basic+）| 好 | PLY/OBJ/FBX/USDZ等 |
| 测量工具 | ✓（Business+）| 好 | — |
| API 接入 | ✓（Business+）| 中 | — |
| 团队协作 | ✓（Business）| 中 | — |
| 移动端（采集+查看）| ✓ | 好 | 主要使用场景 |

#### 定价

| 层级 | 价格 | 核心功能 |
|------|------|---------|
| Free | 免费 | 有限次采集，仅 GLTF 导出，带水印 |
| Basic | $30/月 或 $150/年（$12.5/月）| 无限采集，15+格式导出，3DGS |
| Business | $400/年/用户（$34/月/用户）| 楼层图，测量，团队库，点云导出 |
| Enterprise | 定制 | 专属支持，更高隐私，管理控制 |

**采集限制**：Free≤150张/模型；Basic≤300张/模型；Business/Enterprise≤2000张/模型

#### 用户真实评价

**正面反馈**：
- "最简单的3D扫描应用，学习曲线极低"
- "LiDAR模式扫一个房间只需2-3分钟"
- "导出格式多，和下游软件兼容性好"

**负面反馈**：
- "无LiDAR手机（非Pro iPhone）扫描质量差很多"
- "3DGS效果相比专业工具差一截"
- "没有编辑功能，扫出来的模型无法修改"
- "Business版贵，中小用户难以承受"

#### 优弱势分析

| 核心优势 | 核心弱势 |
|---------|---------|
| 用户规模最大，品牌认知度高 | 强依赖 LiDAR，普通手机体验差 |
| 采集→分享全流程闭环 | 3DGS质量不是最优 |
| 15+格式导出，生态兼容性好 | 无编辑能力 |
| 楼层图、测量等专业功能 | Business 版价格门槛高 |

#### 差异化切入点

> 相对于 Polycam，我们的机会：**普通手机高保真重建**（Polycam在无LiDAR设备上体验差）；**AI补全/编辑能力**（Polycam扫描后无法编辑）；**面向中国及亚洲市场的本地化**。

**正面反馈**：
- "做梦都没想到功能这么强的工具完全免费"

**负面反馈**：
- "不能自己采集，必须配合其他工具"
- "超大场景在低端设备上加载慢"
- "注释/导览功能还不够成熟"

#### 优弱势分析

| 核心优势 | 核心弱势 |
|---------|---------|
| 完全免费，开源，无功能限制 | 无采集能力，是工作流中后段工具 |
| 编辑能力最强（压缩/选择/清理）| 不是全流程平台 |
| 流式 LOD，大场景性能好 | 移动端编辑体验有限 |
| 与主流采集工具（Polycam/Luma）配合使用 | 商业变现模式不清晰 |

#### 差异化切入点

> SuperSplat 是工具，不是平台。相对于 SuperSplat，我们的机会：**全流程一体化**（采集→重建→编辑→分享→协作），用户不需要在多个工具间切换。SuperSplat 反而可以是我们的**生态合作伙伴**——用户可以把 Aholo3D 重建的 PLY 导入 SuperSplat 做精细编辑。

## 竞品格局结论

### 当前市场空白

1. **普通手机 + 高保真重建**：Polycam 强依赖 LiDAR，普通 Android/iPhone 用户体验极差；World Labs 是生成式，精度不可控；市场上没有"普通手机→高精度真实场景"的完整解决方案。

2. **采集→编辑→分享一体化**：Polycam 采集强但无编辑；SuperSplat 编辑强但无采集；World Labs 生成强但场景不真实；没有一个工具做到全流程闭环且面向大众用户。

3. **亚洲/中文市场本地化**：三个竞品均以英语市场为核心，对中国房产、文旅、电商场景缺乏针对性功能（如微信分享、国内支付、本土合规）。

### 我们的差异化定位

**定位一句话**：  
> 在"要么需要专业设备，要么是凭空生成"的3D空间市场中，Aholo3D 是唯一能让普通手机用户30秒扫描→生成高保真真实场景→直接分享/嵌入的一体化平台，专为内容创作者、设计师和中小企业设计。

### 风险提示

**如果 Polycam 在6个月内推出"AI补全"来支持无LiDAR手机，我们还有什么？**
- 中文本地化 + 亚洲市场先发优势
- 更强的编辑能力（参考 SuperSplat 路径）
- 社区/PLG 传播机制（Polycam 是工具而非社区）
- 更低的定价（面向创作者和个人用户）

**如果 World Labs 推出"从现实视频精确重建"功能，我们还有什么？**  
- 移动端完整体验（World Labs 专注 AI 生成，移动端采集不是其方向）
- 面向专业工作流的测量/平面图/BIM集成（更接近 Polycam Business）
- 亚洲市场 + 本地化
