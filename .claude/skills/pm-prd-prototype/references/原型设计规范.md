# Aholo3D 原型设计规范

> 所有由此Skill生成的HTML原型必须严格使用本文档定义的Design Token。  
> 禁止使用规范外的色值、字体、圆角值。禁止引入外部CSS框架。

## 1. 基础规范

### 1.1 Color — 颜色系统

> 以下 Token 从 aholo3d.cn 生产环境提取（2026年4月），为真实值，直接复制即可使用。

#### 主题：暗色（Dark Theme，默认，`.dark` 类启用）

```css
/* ===================================================
   Aholo3D Design Tokens — Dark Theme (生产环境真实值)
   来源：https://www.aholo3d.cn 生产 CSS 提取
   =================================================== */
:root {
  /* === 背景层级（从 body computed: rgb(27,27,29) 确认）=== */
  --background:          #1b1b1d;          /* 页面根背景 body */
  --card:                #1b1b1d;          /* 卡片背景（与根背景同色）*/
  --popover:             rgba(33,33,35,.3);/* 弹窗/Dropdown 背景（带模糊感）*/
  --muted:               rgba(33,33,35,.4);/* 次要背景，禁用区域 */
  --accent:              rgba(255,255,255,.08); /* hover 高亮底色 */

  /* === 前景/文字颜色 === */
  --foreground:          #ffffff;                      /* 主文本 */
  --card-foreground:     #ffffff;                      /* 卡片内文本 */
  --popover-foreground:  #ffffff;                      /* 弹窗内文本 */
  --muted-foreground:    rgba(255,255,255,.6);          /* 次要文本（说明、时间戳）*/
  --accent-foreground:   #ffffff;                      /* hover 状态文字 */

  /* === 主品牌色 === */
  --primary:             #ffffff;                      /* 主色文字/图标（白色系）*/
  --primary-foreground:  #1b1b1d;                      /* 主色按钮上的文字 */
  /* 蓝色 CTA 按钮（从截图观察）*/
  --color-cta:           #3b82f6;                      /* 免费试用等主 CTA */
  --color-cta-hover:     #2563eb;
  --secondary:           rgba(255,255,255,.1);          /* 次级按钮背景 */
  --secondary-foreground:rgba(255,255,255,.9);          /* 次级按钮文字 */

  /* === 边框/分割线 === */

  --input:               rgba(255,255,255,.2);          /* 输入框边框 */
  --ring:                #ffffff;                      /* focus ring */

  --destructive:         #e62323;                      /* 错误/危险 */
  --destructive-subtle:  rgba(230,35,35,.15);

  /* === 图表/标签色（从生产提取）=== */
  --chart-1:  #f05100;   /* 橙红 */
  --chart-2:  #009588;   /* 青绿（成功态）*/
  --chart-3:  #104e64;   /* 深蓝 */
  --chart-4:  #fcbb00;   /* 金黄（警告）*/
  --chart-5:  #f99c00;   /* 橙黄 */

  /* === 侧边栏专属 === */
  --sidebar:                 rgba(0,0,0,0);            /* 侧边栏背景（透明叠在根背景）*/
  --sidebar-foreground:      rgba(255,255,255,.9);
  --sidebar-primary:         #ffffff;
  --sidebar-primary-foreground: #1b1b1d;
  --sidebar-accent:          rgba(255,255,255,.1);      /* 侧边栏激活项背景 */
  --sidebar-accent-foreground: #ffffff;
  --sidebar-border:          rgba(255,255,255,.2);
  --sidebar-ring:            #ffffff;

  /* === 语义语法糖（方便组件引用）=== */
  --color-success:        #009588;                     /* = chart-2，成功态 */
  --color-success-subtle: rgba(0,149,136,.15);
  --color-warning:        #f99c00;                     /* = chart-5，警告 */
  --color-warning-subtle: rgba(249,156,0,.15);
  --color-danger:         #e62323;                     /* = destructive */
  --color-danger-subtle:  rgba(230,35,35,.15);
  --bg-overlay:           rgba(0,0,0,.6);              /* 模态遮罩 */

  /* === 圆角（使用全局 --radius 基准）=== */
  --radius: .625rem;   /* = 10px，生产值 */

```

**颜色使用规则总结（从截图归纳）**：

| 元素 | 背景 | 文字 |
|------|------|------|
| 页面根 | `#1b1b1d` | `#ffffff` |
| 导航栏/顶栏 | 透明（叠根背景）| `#ffffff` |
| 卡片 | `#1b1b1d`（与根同）| `#ffffff` |
| Dropdown/弹窗 | `rgba(33,33,35,.3)` + backdrop-blur | `#ffffff` |
| 主 CTA 按钮（免费试用/创建）| `#3b82f6` | `#ffffff` |
| 次级按钮 | `rgba(255,255,255,.1)` | `rgba(255,255,255,.9)` |
| 激活侧边栏项 | `rgba(255,255,255,.1)` | `#ffffff` |
| 次要文字（时间/说明）| — | `rgba(255,255,255,.6)` |
| 边框/分割线 | — | `rgba(255,255,255,.2)` |

### 1.2 Typography — 字体规范

```css
:root {
  /* === 字体家族（生产实测）=== */
  --font-sans:  'Poppins', 'Arial', sans-serif;  /* 生产字体，英文用 Poppins，中文回退 Arial */
  --font-mono:  'JetBrains Mono', 'Fira Code', monospace;

  /* === 字号 === */
  --text-xs:   11px;   /* 辅助标注、角标 */
  --text-sm:   12px;   /* 说明文、时间戳、标签文字 */
  --text-base: 13px;   /* 正文、列表项（UI主力字号）*/
  --text-md:   14px;   /* 较重要正文、按钮文字 */
  --text-lg:   15px;   /* 面板标题、小节标题 */
  --text-xl:   17px;   /* 模块标题 */
  --text-2xl:  20px;   /* 页面标题 */
  --text-3xl:  24px;   /* 大标题 */
  --text-4xl:  32px;   /* Hero标题 */

  /* === 字重 === */
  --font-regular:  400;
  --font-medium:   500;
  --font-semibold: 600;
  --font-bold:     700;

  /* === 行高 === */
  --leading-tight:  1.3;
  --leading-normal: 1.5;
  --leading-loose:  1.75;
}
```

**字体组合规则**：

| 场景 | 字号 | 字重 | 颜色 |

### 1.3 Corner — 圆角规范

```css
:root {
  --radius-l:  999px;  /* L级：特殊一级按钮（完全圆形，如"Download"主CTA）*/
  --radius-m:  12px;   /* M级：大组件、大卡片、弹窗、Dropdown面板 */
  --radius-s:  10px;   /* S级：工具一级按钮（特殊视觉统一场景）*/
  --radius-xs: 8px;    /* XS级：普通按钮、输入框、小卡片 */
  --radius-ex: 4px;    /* EX级：小按钮、标签、徽标、角标 */
}
```

**圆角使用规则**：

| 组件 | 圆角 |
|------|------|
| 主CTA按钮（Download/Create等）| `--radius-l` 或 `--radius-xs` |
| 模态框/弹窗 | `--radius-m` |
| 卡片 | `--radius-m` |
| Dropdown面板 | `--radius-m` |
| 普通按钮 | `--radius-xs` |
| 输入框 | `--radius-xs` |
| 标签/Tag | `--radius-ex` |
| Pro/Beta徽标 | `--radius-ex` |

> 同一区域内，外包围区圆角12px，内按钮圆角8px（保持视觉层级嵌套感）

## 2. 组件规范

### 2.1 Button — 按钮

#### 按钮层级

| 类型 | 背景 | 文字色 | 边框 | 用途 |
|------|------|--------|------|------|
| Primary | `--color-primary` | `#fff` | 无 | 最重要的CTA，每屏最多1个 |
| Secondary | `--bg-element` | `--text-primary` | `--border-default` | 次要操作 |
| Ghost | 透明 | `--text-secondary` | `--border-default` | 辅助操作 |
| Danger | `--color-danger-subtle` | `--color-danger` | `--color-danger` 20% | 删除、危险操作 |
| Link | 透明 | `--color-primary` | 无 | 文字链接 |

#### 按钮尺寸

| 尺寸 | 高度 | 横向Padding | 字号 | 圆角 |
|------|------|------------|------|------|
| Large (L) | 44px | 20px | `--text-md` | `--radius-xs` 或 `--radius-l` |
| Medium (M) | 36px | 16px | `--text-md` | `--radius-xs` |
| Small (S) | 32px | 12px | `--text-base` | `--radius-xs` |
| XSmall (XS) | 28px | 10px | `--text-sm` | `--radius-ex` |

#### 按钮状态

```css
/* Primary Button */
.btn-primary {
  background: var(--color-primary);
  color: #fff;
  border: none;
  border-radius: var(--radius-xs);
  font-weight: var(--font-medium);
  cursor: pointer;
  transition: all 0.15s ease;
}
.btn-primary:hover  { background: var(--color-primary-hover); }
.btn-primary:active { background: var(--color-primary-active); transform: scale(0.98); }
.btn-primary:disabled { opacity: 0.4; cursor: not-allowed; }
```

## 4. 原型生成规则

### 4.1 HTML模板头部（必须复制）

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>[页面标题] - Aholo3D</title>
  <style>
    /* 必须在此处内联所有CSS变量（从本规范完整复制:root块）*/
    :root { /* ... 所有变量 ... */ }
    * { margin: 0; padding: 0; box-sizing: border-box; }
    body {
      font-family: var(--font-sans);
      background: var(--background);   /* #1b1b1d */
      color: var(--foreground);         /* #ffffff */
      min-height: 100vh;
    }
  </style>
</head>
<body>
  <!-- 页面内容 -->
  <script>
    /* 所有交互逻辑内联在此 */
  </script>
</body>
</html>
```

### 4.2 图片/图标使用规则

- 禁止使用外部图片URL（网络不通时会破坏原型）
- 3D场景缩略图：用CSS渐变代替（参考已有HTML文件中的 `linear-gradient` 示例）
- 图标：使用Unicode符号或内联SVG，禁止引用外部图标库

### 4.3 多页面切换

```javascript
// 单文件多视图切换模式
function showView(viewId) {
  document.querySelectorAll('.view').forEach(v => v.style.display = 'none');
  document.getElementById(viewId).style.display = 'flex'; // 或 'block'
}
```

### 4.4 原型交互说明注释规范

```html
<!-- [交互说明] 点击此按钮触发上传流程，切换到 upload-view -->
<button onclick="showView('upload-view')" class="btn-primary">
  开始扫描
</button>
```

### 4.5 响应式断点与移动端适配规则

| 断点 | 适配场景 |
|------|---------|
| ≥1280px | 桌面端（默认设计基准，**所有页面**）|
| 768px - 1279px | 平板（如PRD明确要求）|
| ≤767px | 移动端（**仅 Viewer 查看器页面**）|

**移动端适配范围明确规定**：

| 页面类型 | 是否需要移动端（≤767px）| 说明 |
|---------|----------------------|------|
| 3D Viewer（查看器/分享页）| **必须适配** | 用户通过分享链接在手机上查看场景，核心 PLG 场景 |
| Library（项目列表）| 可选（PRD注明时才做）| |
| 编辑器（3DGS编辑/工具栏）| **不需要** | 编辑操作复杂度高，仅桌面端 |
| 上传流程 | 可选 | |
| 登录/注册 | 建议适配 | |

## 5. 背景使用规范

| 层级 | 变量 | 使用场景 |
|------|------|---------|
| 根背景 | `--bg-root` | body、页面底层、侧边栏之外的空白区 |
| 主面板 | `--bg-surface` | 卡片、侧边栏、顶栏 |
| 次面板 | `--bg-surface-2` | 弹窗、Dropdown、悬浮面板 |
| 元素 | `--bg-element` | 输入框、选择器、工具按钮背景 |
| 元素hover | `--bg-element-hover` | 列表项hover、选择器hover |

> 背景层级越深（数字越大），颜色越浅——这是暗色UI的核心规律，始终保持此层级关系。
